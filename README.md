# [Blog](https://colinrcummings.github.io/blog/) built with [Phenomic](https://github.com/MoOx/phenomic) (version 1.0.0-beta.3)

## Install dependencies

```sh
npm install
```

## Run development server

```sh
npm start
```

## Build for production

```sh
npm run build
```

## Deploy to GitHub Pages

```sh
npm run deploy
```
